1. Phiên bản v01_01: Cài đặt hàm tính toán sin(x), cos(x), sqrt(x)
- Nội dung:
  + chữa bài tập
  + để xây dựng chương trình/phầm mềm, cần kiến thức lập trình/tin học VÀ 
  cần kiến thức của lĩnh vực (domain) liên quan (ở đây là Giải tích/Phương pháp tính).
- Kỹ thuật:
  + vòng lặp for và while
  + hàm stod() chuyển từ string thành double, kiểu dữ liệu double (các bài khác dùng toàn int)
  + cầu trúc switch không chạy với string (do string không là hằng)
  + trong hàm sqrt(), điều kiện lặp là khi kết quả 2 lần tính gần nhất bằng nhau
  (lý do là dù có sai số, nhưng do (giới hạn chính xác) biểu diễn của máy tính nên vẫn chạy)
